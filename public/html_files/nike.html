Shadows / Elevation
◻
No shadow tokens defined
Nike's design token file returns an empty shadow: {} object. Nike.com uses no CSS custom property-based elevation. Depth is communicated entirely through achromatic contrast, border presence, and layered photography — zero drop shadows or blur anywhere in the system. This is one of only two systems in this series with no shadow tokens (alongside Mistral AI).

Opacity
Opacity Tokens
1 token — 0.85 only
--opacity-85
85% · ×1
The rarest opacity vocabulary in this entire series — a single token at 85%. Used exclusively for the semi-transparent navigation overlay that appears when the mega-menu opens, darkening product imagery without fully obscuring the page. Everything else is fully opaque or fully transparent.
