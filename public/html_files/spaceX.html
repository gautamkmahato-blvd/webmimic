The single border width in the SpaceX design system — used for all dividers, component outlines, and UI boundaries via --border-width-hairline: 1px. Borders use rgba(240,240,250,0.35) on black backgrounds.
Shadow / Elevation
No shadow tokens defined

SpaceX's design system contains no shadow or elevation tokens. The site achieves depth exclusively through high-contrast borders (rgba(240,240,250,0.35)), full-bleed imagery, and tonal separation between pure black backgrounds and video/photo content. This is an intentional aesthetic choice consistent with SpaceX's stark, cinematic visual language.

Opacity
No opacity tokens defined

SpaceX does not define discrete opacity tokens in its design system. Instead, alpha transparency is embedded directly into the color values: --white-10 through --white-100 are pre-composed rgba() colors declared in :root. The opacity ramp is shown as color swatches in the Color Palette section above.
