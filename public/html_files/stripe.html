Breakpoints
No breakpoints defined

Stripe's design system does not expose responsive breakpoint tokens in its CSS custom properties. The breakpoint object in the DTCG JSON export is empty. Stripe's responsive layout is handled through CSS media queries defined directly in component stylesheets rather than through design token variables.
